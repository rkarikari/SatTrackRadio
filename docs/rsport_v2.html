HF Communication Exercises

Ionospheric Propagation

HF signals reflect off the ionosphere, enabling long-distance communication. Key factors:

  • Solar Activity: Track solar flux index for band predictions
  • Time of Day: D-layer absorption during day, F2 layer at night
  • Frequency Selection: Higher frequencies during solar maximum
  • Skip Distance: Minimum distance for sky wave return

Practical HF Projects

  1. Monitor solar flux and predict propagation conditions
  2. Attempt contacts on 20m, 40m bands
  3. Compare dipole vs. vertical antenna performance
  4. Build AI model to forecast optimal operating times

Spectrum Analyzer Example

import numpy as np import matplotlib.pyplot as plt from rtlsdr import RtlSdr class SpectrumAnalyzer: def __init__(self, center_freq=145e6): self.sdr = RtlSdr() self.sdr.center_freq = center_freq self.sdr.sample_rate = 2.4e6 def capture_spectrum(self): samples = self.sdr.read_samples(256*1024) fft_result = np.fft.fftshift(np.fft.fft(samples)) power_db = 10 * np.log10(np.abs(fft_result)**2) return power_db

class FIRFilter { private: std::vector<float> coefficients; std::vector<float> buffer; int bufferIndex; public: float process(float input) { buffer[bufferIndex] = input; float output = 0.0f; int idx = bufferIndex; for (size_t i = 0; i < coefficients.size(); ++i) { output += coefficients[i] * buffer[idx]; idx = (idx == 0) ? buffer.size()-1 : idx-1; } bufferIndex = (bufferIndex + 1) % buffer.size(); return output; } };
